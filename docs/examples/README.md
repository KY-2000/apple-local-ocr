This folder contains copy-paste examples for integrating `apple-local-ocr` into OpenClaw-style agent workflows.

- [openclaw-task-snippets.md](/Users/kenyuen/Desktop/coding-project/apple-local-ocr/docs/examples/openclaw-task-snippets.md): copy-paste command patterns and integration snippets

Recommended order:

1. Read [openclaw-integration.md](/Users/kenyuen/Desktop/coding-project/apple-local-ocr/docs/openclaw-integration.md)
2. Use the wrapper if you want one stable command surface for OpenClaw
3. Adapt the snippets to your own OpenClaw tool or task schema
